# Fonte da Sabedoria

Landing page premium para apoio direto ao projeto Fonte da Sabedoria e ponte para o eBook **O Renascimento da Sua Melhor Versao**.

## Estrutura

- `index.html` - conteudo e estrutura da landing
- `styles.css` - identidade visual, responsivo e animacoes
- `script.js` - animacoes leves de reveal no scroll
- `assets/` - imagens do hero, bloco emocional e mockup do eBook

## Rodar localmente

```bash
python -m http.server 4177 --bind 127.0.0.1
```

Depois acesse:

```text
http://127.0.0.1:4177
```
